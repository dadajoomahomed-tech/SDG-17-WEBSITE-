## Project Description
This project is a multi-page website showcasing **Sustainable Development Goal 17 (Partnerships for the Goals)**.  
It demonstrates how governments, organizations, communities, and individuals can collaborate to achieve global sustainability objectives.  

The website combines informative content, interactive elements, and visual storytelling to raise awareness about SDG 17 and encourage participation in partnerships.

---

## Features
- **Home Page** with hero section, introduction, key SDG 17 principles, image gallery, and call-to-action.  
- **Gallery Page** with image carousel showcasing real-world SDG 17 partnerships.  
- **About Page** explaining the project purpose, focus areas, and development approach.  
- **Contact Form** to propose partnership ideas (with validation and feedback messages).  
- **Responsive Design** for mobile, tablet, and desktop devices.  
- **Accessible Navigation** with semantic HTML and smooth scrolling.  

---

## Technologies Used
- **HTML5** – Semantic markup for structure and accessibility.  
- **CSS3** – Styling, responsive layouts, and interactive elements.  
- **JavaScript** – Hero fade-in, form validation, smooth scrolling, and image carousel.  
- **Images & Assets** – Logos, SDG wheel, gallery images.

## How to View the Website

### Option 1: GitHub Pages
1. Open the repository on GitHub.  
2. Go to **Settings → Pages**.  
3. Select the `main` branch and save.  
4. Open the generated URL to view the live site.

### Option 2: Local Machine
1. Clone or download the repository.  
2. Open `index.html` in any modern web browser.  
3. Navigate using the menu to view other pages.
